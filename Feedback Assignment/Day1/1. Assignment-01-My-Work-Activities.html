2Prompt ພ້ອມໃຊ້ — ສຳເນົາ ແລະ ດັດແກ້ · Ready-to-use prompt

Copy ກ່ອງລຸ່ມນີ້ ໃສ່ Cowork. ປ່ຽນ ສ່ວນ ສີເຫຼືອງ ໃຫ້ ກົງກັບ ຕົວທ່ານ.

PROMPT — ສຳເນົາທັງໝົດ ສະບາຍດີ Claude, ຂ້ອຍແມ່ນ [ຊື່-ນາມສະກຸນ] ເຮັດຕຳແໜ່ງ [ຕຳແໜ່ງ] ຢູ່ EMI (ສະຖາບັນການເງິນຈຸລະພາກ ເອັກພັດທະນາ — Ekphatthana MFI, ສປປ ລາວ). ຂະແໜງ/ພະແນກ ຂອງຂ້ອຍ ແມ່ນ [ຂະແໜງ/ພະແນກ]. ມື້ນີ້ ຂ້ອຍ ກຳລັງ ຝຶກ ໃຊ້ Claude ເປັນຄັ້ງທຳອິດ. ຂ້ອຍຢາກ ໃຫ້ ເຈົ້າ ຮູ້ຈັກ ຂ້ອຍ ກ່ອນ ໂດຍຜ່ານ ການອະທິບາຍ ວຽກ ປະຈຳວັນ, ປະຈຳອາທິດ, ແລະ ປະຈຳເດືອນ ຂອງຂ້ອຍ. ກະລຸນາ ຖາມ ຄຳຖາມ ຂ້ອຍ ເທື່ອລະ 2-3 ຄຳຖາມ ເພື່ອ ເຂົ້າໃຈ: 1) ວຽກ ປະຈຳວັນ — ກິດຈະກຳ ທີ່ ຂ້ອຍ ເຮັດ ທຸກວັນ 2) ວຽກ ປະຈຳອາທິດ — ສິ່ງທີ່ ເຮັດ 1-2 ຄັ້ງ ຕໍ່ ອາທິດ 3) ວຽກ ປະຈຳເດືອນ — ບົດລາຍງານ ຫຼ ໜ້າວຽກ ທີ່ ມາ ປະຈຳເດືອນ 4) ສິ່ງທີ່ ຍາກ ຫຼ ໃຊ້ ເວລາ ຫຼາຍ 5) ສິ່ງທີ່ ຂ້ອຍ ຫວັງ ໃຫ້ Claude ຊ່ວຍ ກະລຸນາ ສົນທະນາ ກັບ ຂ້ອຍ ເປັນ ພາສາລາວ ທັງໝົດ. ເມື່ອ ເຮົາ ຄຸຍ ສຳເລັດ, ກະລຸນາ ສ້າງ ໄຟລ໌ HTML ໜຶ່ງ ໜ້າ ໃນ ຮູບແບບ EMI ໂດຍ ໃຊ້: - ສີ EMI Green: #49C18E - ສີ Navy Blue: #17335E - ສີ ພື້ນຫຼັງ: ຂາວ - font: Noto Sans Lao - ມີ ສ່ວນຫົວ (header) ບອກ ຊື່, ຕຳແໜ່ງ, ພະແນກ - ມີ 3 ສ່ວນ: ປະຈຳວັນ / ປະຈຳອາທິດ / ປະຈຳເດືອນ - ມີ ສ່ວນທ້າຍ "ສິ່ງທີ່ Claude ສາມາດ ຊ່ວຍ" - ໃຊ້ ສຳນວນ ສຸພາບ ແບບ ທາງການ EMI ເລີ່ມ ໂດຍ ການ ຖາມ ຄຳຖາມ ຊຸດທີ 1 (ວຽກ ປະຈຳວັນ).

ເຄັດລັບ: ຖ້າ Claude ຕອບ ເປັນ ພາສາອັງກິດ, ໃຫ້ ບອກ ມັນ: "ກະລຸນາ ຕອບ ເປັນ ພາສາລາວ ທັງໝົດ"

3ສູດ EMI Prompt · The EMI prompt formula

ທຸກ prompt ທີ່ ດີ ມີ 5 ສ່ວນ. ຈົດ ໃສ່ ໃຈ ສະເໝີ:

[ ຂ້ອຍແມ່ນໃຜ ] + [ ຕ້ອງການຫຍັງ ] + [ ສຳລັບໃຜ ] + [ ຮູບແບບ ] + [ ພາສາ ]
WHOຂ້ອຍແມ່ນ [ຕຳແໜ່ງ] ຢູ່ EMI
WHATຂ້ອຍຢາກ ໃຫ້ ເຈົ້າ ສ້າງ / ຂຽນ / ສະຫຼຸບ / ແປ...
AUDIENCEສຳລັບ ຫົວໜ້າ / ລູກຄ້າ / ບອດ / ສະຕາຟ
FORMATເປັນ HTML / ຈົດໝາຍ / ບົດລາຍງານ / Facebook post
LANGUAGEເປັນ ພາສາລາວ ທາງການ ຫຼ ພາສາອັງກິດ
