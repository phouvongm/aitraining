3ຈຸດປະສົງຂອງຕຳແໜ່ງ · Job Purpose

ນຳພາ ພະແນກເຕັກໂນໂລຊີຂໍ້ມູນຂ່າວສານຂອງ EMI ໃນການວາງແຜນ, ບໍລິຫານຈັດການ ແລະ ສະໜອງໂຄງລ່າງດິຈິຕ້ອນ ທີ່ມີສະຖຽນລະພາບ, ປອດໄພ ແລະ ສາມາດຮອງຮັບການເຕີບໃຫຍ່ຂອງສະຖາບັນ.

ຮັບປະກັນ ວ່າລະບົບເຕັກໂນໂລຊີຫຼັກ (Optimal Core Banking, Odoo ERP, AWS Cloud, Network, Email) ສະໜັບສະໜູນຍຸດທະສາດ EMI Pathway 2030 ແລະ ສອດຄ່ອງກັບມາດຕະຖານການກວດກາຂອງທະນາຄານແຫ່ງ ສປປ ລາວ (BOL).

ປົກປ້ອງ ຂໍ້ມູນຂອງລູກຄ້າ ແລະ ສະຖາບັນຈາກໄພຄຸກຄາມທາງໄຊເບີ ໂດຍຜ່ານກອບຄວາມປອດໄພ NIST CSF ແລະ ການທົດສອບແຜນກູ້ຄືນລະບົບ (DRP) ປະຈຳປີ ເພື່ອຮັບປະກັນຄວາມຕໍ່ເນື່ອງຂອງທຸລະກິດ.

4ຄວາມຮັບຜິດຊອບຫຼັກ · Key Responsibilities

ໜ້າທີ່ຫຼັກ 8 ດ້ານ ໂດຍແບ່ງ % ນ້ຳໜັກວຽກ ລວມ = 100%:

  1. ບໍລິຫານ ໂຄງລ່າງ IT ແລະ ການເຮັດວຽກປະຈຳວັນ (25%) ກວດສອບ ສຸຂະພາບລະບົບປະຈຳວັນ (Daily Health Check) ຂອງເຄືອຂ່າຍ, AWS EC2, Windows Servers; ຄຸ້ມຄອງ Backup ປະຈຳວັນ ແລະ ການກູ້ຄືນຂໍ້ມູນ. KPI: Uptime ≥ 99.5%/ເດືອນ, Backup success ≥ 99%.
  2. ປົກປ້ອງ ຄວາມປອດໄພທາງໄຊເບີ ແລະ ບໍລິຫານຄວາມສ່ຽງ (20%) ປະຕິບັດ ກອບຄວາມປອດໄພ NIST CSF Lite; ດຳເນີນ Vulnerability Assessment 2 ຮອບ/ປີ; ຝຶກອົບຮົມ Security Awareness ສຳລັບພະນັກງານທັງໝົດ; ລາຍງານ ເຫດການທີ່ກ່ຽວຂ້ອງຕໍ່ Risk Committee ແລະ BOL.
  3. ບໍລິການ IT Helpdesk ແລະ ສະໜັບສະໜູນຜູ້ໃຊ້ (15%) ຄຸ້ມຄອງ IT Helpdesk ສຳລັບ 50–80 ຜູ້ໃຊ້ + 70–100 devices; ອະນຸມັດ ສິດການເຂົ້າເຖິງ (User Access Control); ບໍລິຫານ ລະບົບ Email; ແກ້ໄຂ ບັນຫາທີ່ສັບສົນ. KPI: Response ≤ 2 ຊມ, Resolution ≤ 24 ຊມ, CSAT ≥ 4.0/5.
  4. ນຳພາ ໂຄງການຍຸດທະສາດ IT — Pathway 2030 (20%) ດຳເນີນ 3 Strategic Priorities ໃນປີ 2026: (1) AWS Cloud Optimization & DRP Maturity, (2) Cybersecurity Framework Implementation, (3) Digital Channel Foundation (API-ready integration: Optimal ↔ Odoo ↔ Digital channels).
  5. ບໍລິຫານ ງົບປະມານ, ການຈັດຊື້ ແລະ ການປະຕິບັດຕາມກົດໝາຍ (10%) ວາງແຜນ ງົບປະມານ IT ປະຈຳປີ; ກວດສອບ ຄ່າໃຊ້ຈ່າຍ AWS (ເປົ້າຫຼຸດ 15%/ປີ); ສ້າງ ບົດລາຍງານປະຈຳເດືອນຕໍ່ Board; ປະສານ ການກວດສອບ BOL.
  6. ນຳພາ ທີມງານ ແລະ ພັດທະນາບຸກຄະລາກອນ (10%) ບໍລິຫານ ທີມ IT 2–4 ຄົນ; ຈັດ Stand-up Meeting ປະຈຳເຊົ້າ; ປະເມີນ ຜົນງານ; ວາງແຜນ Upskilling ≥ 40 ຊມ/ຄົນ/ປີ.
  7. ປະສານງານ ກັບ ຄູ່ຮ່ວມງານພາຍນອກ ດ້ານກົດໝາຍ, ການເງິນ, ໂທລະຄົມ ແລະ ໂຄງການ ປະສານ ກັບ BOL, ກົມຊຳລະ, ກົມເຕັກໂນໂລຊີ, JITS, IT Technical Consultant, BCEL, LDB, ETL, LTC, Unitel ແລະ ICCAP ໃນການລາຍງານ, ການເຊື່ອມຕໍ່ລະບົບ, ການຊຳລະເງິນ, ການເຊື່ອມຕໍ່ Internet/MPLS ແລະ ໂຄງການຄູ່ຮ່ວມງານ.
  8. ທົດສອບ ແຜນກູ້ຄືນລະບົບ (DRP Testing) ປະຈຳໄຕມາດ — 4 ຄັ້ງ/ປີ ທົບທວນ ແລະ ທົດສອບ Disaster Recovery Plan ທຸກໆ 3 ເດືອນ; ກວດສອບ ການ Failover; ບັນທຶກ ຜົນການທົດສອບເພື່ອລາຍງານ Risk Committee ແລະ BOL ຕາມມາດຕະຖານ Business Continuity Plan.
ໝາຍເຫດ: ທຸກຄຳກຳມະທີ່ໝາຍສີຂຽວ (ກວດສອບ, ບໍລິຫານ, ນຳພາ, ປະຕິບັດ, ປະສານງານ ...) ແມ່ນ action verbs ສະເພາະ — ບໍ່ໃຊ້ຄຳກວ້າງເຊັ່ນ "ຊ່ວຍ" ຫຼື "ສະໜັບສະໜູນ".

5ຄຸນສົມບັດທີ່ຕ້ອງການ · Qualifications

ການສຶກສາ, ໃບປະກາດ ແລະ ປະສົບການທີ່ຕ້ອງການສຳລັບຕຳແໜ່ງນີ້ — ແບ່ງເປັນຂັ້ນຕ່ຳ (Required) ແລະ ສະເພາະ (Preferred).

ຂັ້ນຕ່ຳ · Required

  • ການສຶກສາ: ປະລິນຍາຕີ ສາຂາ Computer Science / Information Technology / Computer Engineering / MIS ຫຼື ສາຂາທີ່ກ່ຽວຂ້ອງ
  • ປະສົບການລວມ: ຢ່າງໜ້ອຍ 8 ປີ ໃນວຽກ IT
  • ປະສົບການຄຸ້ມຄອງ: ຢ່າງໜ້ອຍ 3 ປີ ໃນບົດບາດນຳພາທີມ IT
  • ໃບປະກາດພື້ນຖານ: ITIL Foundation ຫຼື ໃບປະກາດການຄຸ້ມຄອງ IT ທີ່ທຽບເທົ່າ
  • ພາສາ: ລາວ (ດີຫຼາຍ) + ອັງກິດ (Working level — Reading/Writing)
  • ຄວາມຮູ້ດ້ານອຸດສາຫະກຳ: ເຂົ້າໃຈກອບການເງິນຈຸລະພາກ, ກົດໝາຍ BOL ສຳລັບ Deposit-Taking MFI

ສະເພາະ · Preferred

  • ການສຶກສາ: ປະລິນຍາໂທ ສາຂາ IT Management, Cybersecurity, Information Systems ຫຼື MBA
  • ປະສົບການລວມ: 10+ ປີ ໃນວຽກ IT
  • ປະສົບການອຸດສາຫະກຳ: 3+ ປີ ໃນ ທະນາຄານ / MFI / Fintech
  • ໃບປະກາດ Security: CISM ຫຼື CISSP (Information Security Management)
  • ໃບປະກາດ Cloud: AWS Certified Solutions Architect (Associate ຫຼື Professional)
  • ໃບປະກາດ Project: PMP, Prince2 ຫຼື Agile/Scrum Master
  • ພາສາ: ອັງກິດ Business level (ສາມາດນຳພາປະຊຸມກັບ vendors ສາກົນ ແລະ ໂຄງການ ADA, ICCAP)
ໝາຍເຫດ: ໃບປະກາດໃນສ່ວນ "ສະເພາະ" ບໍ່ບັງຄັບໃນຕອນເລີ່ມຕົ້ນ — ແຕ່ EMI ສະໜັບສະໜູນໃຫ້ຜູ້ດຳລົງຕຳແໜ່ງສຶກສາເພີ່ມເຕີມພາຍໃນ 2 ປີ ໂດຍຜ່ານໂຄງການພັດທະນາບຸກຄະລາກອນ.

  • ຫົວໜ້າສາຂາ + ໜ່ວຍບໍລິການ: ການໃຫ້ບໍລິການ IT Support, ການແກ້ໄຂບັນຫາ remote

ພາຍນອກອົງກອນ — ດ້ານກົດໝາຍ ແລະ ກວດກາ REGULATORS

  • ທະນາຄານແຫ່ງ ສປປ ລາວ (BOL): ການລາຍງານ IT, ການກວດສອບການປະຕິບັດຕາມກົດໝາຍ Deposit-Taking MFI
  • ກົມຊຳລະ (Payment System Department — BOL): ການເຊື່ອມຕໍ່ລະບົບຊຳລະເງິນ, ການລາຍງານທຸລະກຳ
  • ກົມເຕັກໂນໂລຊີ (Technology Department — BOL): ການລາຍງານໂຄງລ່າງ IT, ການອະນຸມັດເຕັກໂນໂລຊີໃໝ່

ພາຍນອກອົງກອນ — ໂຄງການ ແລະ ທີ່ປຶກສາ PROJECTS

  • ADA (Appui au Développement Autonome): ໂຄງການ Digital Transformation, SPI4 reporting, ການພັດທະນາ Social Performance Management
  • ICCAP: ໂຄງການຄູ່ຮ່ວມງານ, ການລາຍງານ Social Performance, ການສະໜັບສະໜູນ Digital Transformation
  • JITS: ການປະສານງານດ້ານລະບົບ IT, ການເຊື່ອມຕໍ່ບໍລິການ
  • IT Technical Consultant: ການໃຫ້ຄຳປຶກສາທາງເຕັກນິກ, Architecture review, Security advisory

ພາຍນອກອົງກອນ — ສະຖາບັນການເງິນ ແລະ ໂທລະຄົມ PARTNERS

  • BCEL (ທະນາຄານການຄ້າຕ່າງປະເທດລາວ): ການເຊື່ອມຕໍ່ການຊຳລະເງິນ, Inter-bank transfer, Payment gateway
  • LDB (ທະນາຄານພັດທະນາລາວ): ການເຊື່ອມຕໍ່ການຊຳລະເງິນ, ການຮ່ວມມືດ້ານທຸລະກຳ
  • ETL (ບໍລິສັດໂທລະຄົມລາວ): ການເຊື່ອມຕໍ່ Internet, MPLS, Leased lines ສຳລັບສຳນັກງານໃຫຍ່ ແລະ ສາຂາ
  • LTC (ບໍລິສັດ ລາວໂທລະຄົມ): Backup connectivity, ການເຊື່ອມຕໍ່ສຳຮອງສຳລັບສາຂາຕ່າງແຂວງ
  • Unitel: Mobile data, USSD services, ການເຊື່ອມຕໍ່ສຳຮອງ

ພາຍນອກອົງກອນ — Vendors ແລະ Auditors VENDORS

  • AWS / Cloud Vendor: ການຄຸ້ມຄອງ Cloud Infrastructure, Cost Management, Technical support
  • Optimal Core Banking Vendor: ການບຳລຸງຮັກສາ, ການອັບເດດ, ການແກ້ໄຂບັນຫາ Core Banking
  • External IT Auditors: IT Audit, Cybersecurity Assessment ປະຈຳປີ
